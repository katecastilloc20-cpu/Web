FUNDAMENTOS

Box Model

El Box Model (modelo de caja) en CSS es la forma en que el navegador representa y organiza cada elemento en una página web. Básicamente, todo en HTML es una caja.

Display

La propiedad display en CSS define cómo se comporta un elemento en la página, es decir, cómo se muestra y cómo se organiza con los demás.

Flexbox_1

Flexbox es una forma de organizar elementos en fila o columna y alinearlos fácilmente.

Flexbox_2

Es cuando ya no solo usas Flexbox para poner elementos en fila o centrarlos, sino para controlar cómo se adaptan, crecen, se encogen y se acomodan en diferentes pantallas.

Medidas

Son las medidas que se usan para definir espacios y tamaños

Propiedades

Son las caracteristicas que se aplican en Css como color, tamaño, margen, entre otras

Selectores

Son los elementos que usa Css para definir que se va a modificar el Html
